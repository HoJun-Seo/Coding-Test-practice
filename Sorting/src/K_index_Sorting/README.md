# Sorting
## 문제 개요
- 아래의 링크를 통해 몇번의 반복 실행 동안 배열에서 특정한 영역 만큼을 잘라낸 후 해당 배열을 정렬한 뒤 특정 인덱스에 있는 값이 무엇인지 알아내고, 해당 요소를 각 반복마다 answer 배열에 저장하여 
최종적으로 해당 answer 배열을 return 해주는 프로그램을 작성해보자.
